# 🚀 Série D — Agent v1.0 (Pilotage Mobile)

Système de déploiement automatisé optimisé pour **Samsung A05**. Cet agent utilise l'IA pour transformer une idée en un site web réel hébergé sur Vercel.

## 🛠️ Composants du Système
- **Frontend** : Interface épurée (HTML5/CSS3).
- **Moteur** : Logique de pilotage via `app.js` et **Octokit**.
- **Pipeline** : Plan de repli automatique (Planification -> Code -> Push GitHub).
- **Hébergement** : Configuré pour **Vercel** via `vercel.json`.

## ⚙️ Configuration Rapide
1. Ouvrez l'application via votre lien Vercel.
2. Cliquez sur **☰** (Menu latéral).
3. Entrez vos jetons (GitHub, Groq, Vercel).
4. Décrivez votre projet dans le champ central et lancez la fusée **🚀**.
